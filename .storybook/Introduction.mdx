# Leroy Merlin PT — Component Library

A library of reusable Vue 3 UI components for Leroy Merlin Portugal, built with Tailwind CSS v4.

---

## Getting Started

This library provides a collection of accessible, themeable, and composable UI components built with Vue 3 and Tailwind CSS v4. All packages are published to GitHub Packages under the `@leroy-merlin-pt` npm scope and can be installed individually — only bring in what your project needs.

---

## Installation

### 1. Configure the GitHub Packages registry

Add a `.npmrc` file to the root of your project to point the `@leroy-merlin-pt` scope at the GitHub Packages registry:

```
@leroy-merlin-pt:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR_TOKEN
```

Replace `YOUR_TOKEN` with a GitHub Personal Access Token that has the `read:packages` scope.

### 2. Install individual packages

Install only the components your project requires:

```bash
# npm
npm install @leroy-merlin-pt/button @leroy-merlin-pt/input

# yarn
yarn add @leroy-merlin-pt/button @leroy-merlin-pt/input

# pnpm
pnpm add @leroy-merlin-pt/button @leroy-merlin-pt/input
```

### 3. Set up Tailwind CSS

Components require Tailwind CSS v4. Each package ships a self-contained `style.css` that already includes the Tailwind utilities it needs. Import each component's stylesheet in your project's main entry file:

```js
import '@leroy-merlin-pt/button/style.css'
// Import each component's styles individually as needed
```

Component styles are self-contained — they include all required Tailwind utilities and do not depend on a global Tailwind build in your application.

## Usage

### Basic example

```vue
<script setup>
import Button from '@leroy-merlin-pt/button'
</script>

<template>
  <Button label="Click me" variant="primary" @click="handleClick" />
</template>
```

### With Vue plugin (optional global registration)

You can register components globally if preferred:

```js
import { createApp } from 'vue'
import Button from '@leroy-merlin-pt/button'
import App from './App.vue'

const app = createApp(App)
app.component('Button', Button)
app.mount('#app')
```

## TypeScript Support

All packages ship with TypeScript definitions (`.d.ts` files). No additional `@types/` packages are required — type information is included out of the box.

---

## Browser Support

Modern browsers with ES2015+ support. Internet Explorer 11 is not supported.
