# Leroy Merlin PT — Component Library

A library of reusable Vue 3 UI components for Leroy Merlin Portugal, built with Tailwind CSS v4.

---

## Getting Started

All packages are published to GitHub Packages under the `@leroy-merlin-pt` npm scope. Install only the components your project needs.

### 1. Configure the registry

Add a `.npmrc` file to the root of your project:

```
@leroy-merlin-pt:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR_TOKEN
```

Replace `YOUR_TOKEN` with a GitHub Personal Access Token with the `read:packages` scope.

### 2. Install and import

Each component section below shows the exact install command, CSS import, and a usage example.

## Components

### Button

Interactive button with multiple variants, appearances, sizes, and states. Supports icon slots, loading state, and form submission.

```bash
yarn add @leroy-merlin-pt/button
```

```js
import '@leroy-merlin-pt/button/style.css'
```

```vue
<script setup>
import Button from '@leroy-merlin-pt/button'
</script>

<template>
  <Button label="Confirmar" variant="primary" @click="handleClick" />
</template>
```

**Props:** `label`, `variant` (primary / secondary / ghost), `appearance` (standard / accent / danger / inverse), `size` (medium / small), `disabled`, `isLoading`, `outlined`, `ghost`, `iconPosition`, `type`

### Breadcrumb

Navigation breadcrumb showing the hierarchical path within the application.

```bash
yarn add @leroy-merlin-pt/breadcrumb
```

```js
import '@leroy-merlin-pt/breadcrumb/style.css'
```

```vue
<script setup>
import Breadcrumb from '@leroy-merlin-pt/breadcrumb'

const items = [
  { label: 'Home', href: '/' },
  { label: 'Category', href: '/category' },
  { label: 'Product', href: '/category/product' },
]
</script>

<template>
  <Breadcrumb :items="items" />
</template>
```

**Props:** `items` (array of `{ label, href }`), `appearance` (light / dark)

### Input

Text input field with optional label, hint text, prefix/suffix, and validation states.

```bash
yarn add @leroy-merlin-pt/input
```

```js
import '@leroy-merlin-pt/input/style.css'
```

```vue
<script setup>
import Input from '@leroy-merlin-pt/input'
import { ref } from 'vue'

const value = ref('')
</script>

<template>
  <Input
    v-model="value"
    label="Email"
    placeholder="name@example.com"
    type="email"
  />
</template>
```

**Props:** `modelValue`, `label`, `placeholder`, `id`, `name`, `type`, `size`, `disabled`, `readonly`, `error`, `prefix`, `suffix`, `isClearable`, `clearLabel`

### Modal

Accessible dialog overlay for focused user interactions. Rendered via `<Teleport>` to the document body.

```bash
yarn add @leroy-merlin-pt/modal
```

```js
import '@leroy-merlin-pt/modal/style.css'
```

```vue
<script setup>
import Modal from '@leroy-merlin-pt/modal'
import { ref } from 'vue'

const isOpen = ref(false)
</script>

<template>
  <button @click="isOpen = true">Open modal</button>

  <Modal
    :open="isOpen"
    title="Confirm action"
    description="Are you sure you want to continue?"
    @close="isOpen = false"
  />
</template>
```

**Props:** `open`, `title`, `description`, `closable`, `size` (medium / small), `scroll`, `closeOnOverlay`
**Slots:** `default` (body content), `footer`

**Props:** `id`, `text`, `position` (top / bottom / left / right), `pointer`, `standalone`
**Slots:** `default` (trigger element)

**Props:** `label`, `type` (informative / interactive / selectable / removable / contextualised), `size` (medium / small), `id`, `name`, `modelValue`, `disabled`, `contextualisedNumber`, `removableLabel`

## Design Tokens

The `@leroy-merlin-pt/tokens` package exposes the brand CSS custom properties (colors, typography, spacing) used by all components.

```bash
yarn add @leroy-merlin-pt/tokens
```

```js
import '@leroy-merlin-pt/tokens/style.css'
```

Import once at the root of your application to make all tokens available globally.
