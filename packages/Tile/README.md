# Tile

A tile component for displaying content in a card-like container.

## Installation

```bash
npm install @leroy-merlin-pt/tile
# or
yarn add @leroy-merlin-pt/tile
```

## Basic Usage

```vue
<template>
  <Tile>
    <img src="/product.jpg" alt="Product" />
    <p>Product Name</p>
  </Tile>
</template>
```
