# KpiItem

A KPI (Key Performance Indicator) item component for displaying metrics with a label and value.

## Installation

```bash
npm install @leroy-merlin-pt/kpi-item
# or
yarn add @leroy-merlin-pt/kpi-item
```

## Basic Usage

```vue
<template>
  <KpiItem label="Total Sales" value="€12,500" />
</template>
```
