# FileUploader

A file upload component allowing users to select and upload files via drag-and-drop or file browser.

## Installation

```bash
npm install @leroy-merlin-pt/file-uploader
# or
yarn add @leroy-merlin-pt/file-uploader
```

## Basic Usage

```vue
<template>
  <FileUploader @upload="handleUpload" />
</template>
```
