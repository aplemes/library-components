# TileSelectable

A selectable tile component that acts as a styled radio button or checkbox option.

## Installation

```bash
npm install @leroy-merlin-pt/tile-selectable
# or
yarn add @leroy-merlin-pt/tile-selectable
```

## Basic Usage

```vue
<template>
  <TileSelectable id="tile1" name="choice" value="option1" v-model="selected">
    Option 1
  </TileSelectable>
</template>
```
