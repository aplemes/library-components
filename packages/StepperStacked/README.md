# StepperStacked

A stacked stepper component showing numbered steps in a vertical layout.

## Installation

```bash
npm install @leroy-merlin-pt/stepper-stacked
# or
yarn add @leroy-merlin-pt/stepper-stacked
```

## Basic Usage

```vue
<template>
  <StepperStacked :steps="['Account', 'Details', 'Confirm']" :current="0" />
</template>
```
