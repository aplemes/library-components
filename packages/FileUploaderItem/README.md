# FileUploaderItem

An individual file item within the FileUploader, showing file name, size, and status.

## Installation

```bash
npm install @leroy-merlin-pt/file-uploader-item
# or
yarn add @leroy-merlin-pt/file-uploader-item
```

## Basic Usage

```vue
<template>
  <FileUploaderItem name="document.pdf" :size="204800" status="success" />
</template>
```
