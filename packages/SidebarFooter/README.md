# SidebarFooter

A footer component for use at the bottom of a Sidebar.

## Installation

```bash
npm install @leroy-merlin-pt/sidebar-footer
# or
yarn add @leroy-merlin-pt/sidebar-footer
```

## Basic Usage

```vue
<template>
  <SidebarFooter>
    <span>Version 1.0.0</span>
  </SidebarFooter>
</template>
```
