# Popover

A popover component that displays contextual information or actions in a floating panel triggered by user interaction.

## Installation

```bash
npm install @leroy-merlin-pt/popover
# or
yarn add @leroy-merlin-pt/popover
```

## Basic Usage

```vue
<template>
  <Popover>
    <template #activator><button>Show info</button></template>
    <p>Popover content here.</p>
  </Popover>
</template>
```
