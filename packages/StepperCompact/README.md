# StepperCompact

A compact stepper indicator showing the current step in a multi-step process.

## Installation

```bash
npm install @leroy-merlin-pt/stepper-compact
# or
yarn add @leroy-merlin-pt/stepper-compact
```

## Basic Usage

```vue
<template>
  <StepperCompact :current="2" :total="5" />
</template>
```
