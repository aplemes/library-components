# SidebarHeader

A header component for use at the top of a Sidebar, typically containing a logo or brand name.

## Installation

```bash
npm install @leroy-merlin-pt/sidebar-header
# or
yarn add @leroy-merlin-pt/sidebar-header
```

## Basic Usage

```vue
<template>
  <SidebarHeader logo="/logo.svg" title="My App" />
</template>
```
