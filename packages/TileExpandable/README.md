# TileExpandable

A tile component with expandable/collapsible content sections.

## Installation

```bash
npm install @leroy-merlin-pt/tile-expandable
# or
yarn add @leroy-merlin-pt/tile-expandable
```

## Basic Usage

```vue
<template>
  <TileExpandable title="Product Details">
    <p>Detailed product information.</p>
  </TileExpandable>
</template>
```
