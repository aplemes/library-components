# OptionListbox

A listbox component that provides a scrollable list of selectable options.

## Installation

```bash
npm install @leroy-merlin-pt/option-listbox
# or
yarn add @leroy-merlin-pt/option-listbox
```

## Basic Usage

```vue
<template>
  <OptionListbox :options="options" v-model="selected" />
</template>
```
