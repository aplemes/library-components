# StepperInline

An inline stepper component showing numbered steps in a horizontal layout.

## Installation

```bash
npm install @leroy-merlin-pt/stepper-inline
# or
yarn add @leroy-merlin-pt/stepper-inline
```

## Basic Usage

```vue
<template>
  <StepperInline :steps="['Step 1', 'Step 2', 'Step 3']" :current="1" />
</template>
```
