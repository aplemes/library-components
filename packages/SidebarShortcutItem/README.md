# SidebarShortcutItem

A shortcut item for use within the SidebarShortcuts component.

## Installation

```bash
npm install @leroy-merlin-pt/sidebar-shortcut-item
# or
yarn add @leroy-merlin-pt/sidebar-shortcut-item
```

## Basic Usage

```vue
<template>
  <SidebarShortcutItem label="Settings" icon="settings" href="/settings" />
</template>
```
