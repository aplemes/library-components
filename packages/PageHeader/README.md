# PageHeader

A page header component that provides a consistent top-level page heading with optional actions.

## Installation

```bash
npm install @leroy-merlin-pt/page-header
# or
yarn add @leroy-merlin-pt/page-header
```

## Basic Usage

```vue
<template>
  <PageHeader title="Dashboard">
    <template #actions><button>Export</button></template>
  </PageHeader>
</template>
```
