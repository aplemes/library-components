# TileClickable

A clickable tile component that acts as an anchor or button, linking to a resource or triggering an action.

## Installation

```bash
npm install @leroy-merlin-pt/tile-clickable
# or
yarn add @leroy-merlin-pt/tile-clickable
```

## Basic Usage

```vue
<template>
  <TileClickable href="/product/1">
    <img src="/product.jpg" alt="Product" />
  </TileClickable>
</template>
```
