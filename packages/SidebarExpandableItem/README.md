# SidebarExpandableItem

An expandable navigation item for use within a Sidebar component.

## Installation

```bash
npm install @leroy-merlin-pt/sidebar-expandable-item
# or
yarn add @leroy-merlin-pt/sidebar-expandable-item
```

## Basic Usage

```vue
<template>
  <SidebarExpandableItem label="Products">
    <SidebarNavItem href="/products/all" label="All Products" />
  </SidebarExpandableItem>
</template>
```
