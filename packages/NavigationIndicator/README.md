# NavigationIndicator

A navigation indicator component used to show the current step or page in a multi-step process.

## Installation

```bash
npm install @leroy-merlin-pt/navigation-indicator
# or
yarn add @leroy-merlin-pt/navigation-indicator
```

## Basic Usage

```vue
<template>
  <NavigationIndicator :current="2" :total="5" />
</template>
```
